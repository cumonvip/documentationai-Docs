## Quick Guide

<Steps>
  <Step title="Access dashboard">
    Go to <b>https://dashboard.sim1.com</b> and sign in.
  </Step>
  <Step title="Start a new project">
    Click the <b>"Create Project"</b> button in the top-right corner.
  </Step>
  <Step title="Enter project info">
    Provide a name and brief description for your docs project.
  </Step>
  <Step title="Finish and view">
    Click <b>"Create"</b> to set up. You'll be taken to the project homepage.
  </Step>
</Steps>

<Callout kind="tip">Invite collaborators from the project members tab!</Callout>
