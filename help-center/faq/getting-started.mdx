## Overview

To create a new documentation project in Sim1:

1. Log in to your Sim1 dashboard.
2. Click the **"Create Project"** button.
3. Enter your project's name and description.
4. Choose a privacy setting.
5. Click **"Create"** to finish.

You can now add, manage, or organize your docs in the new project.

<Callout kind="tip">You can edit project details anytime from the project settings page.</Callout>
